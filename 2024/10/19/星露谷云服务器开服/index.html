星露谷云服务器开服

前言：本教程基于阿里云服务器Ubuntu系统，通过Stardew Multiplayer Server Mod创建host机器人保持服务器一直在线，可以满足朋友间的联机需求。

一.搭建阿里云服务器

注册阿里云账号，点击免费试用

免费地域选择中国香港，配置建议选择2核4G 经济型，系统选择Ubuntu,预装应用不需要选择，然后开始试用

点击管理试用，跳转到建立的服务器，点击左侧状态栏中的实例

二.安装Ubuntu桌面图形环境

https://blog.csdn.net/weixin_43436925/article/details/104558680?utm_medium=distribute.pc_relevant.none-task-blog-2~default~baidujs_baidulandingword~default-1.no_search_link&spm=1001.2101.3001.4242.2&utm_relevant_index=4

三.安装Steam,下载星露谷物语

打开火狐浏览器，搜索Steam，至官网进行下载

下载完成后到文件夹中输入Ctrl + Alt + T 打开终端

输入账户密码，等待后输入Y继续执行

至此安装完毕，登录Steam账户下载星露谷物语

四.安装SMAPI，添加多人联机mod

网站https://www.nexusmods.com/stardewvalley/mods/

下载SMAPI模组

选择Extract Here, 解压后打开文件夹

打开星露谷本地文件夹，将解压后文件夹中的内容拖至其中，运行install on Linux.sh文件

执行中，需复制启动地址，粘贴在启动项选择中

下载Stardew Multiplayer Server Mod模组，解压后将其置入Mods文件夹中

https://www.nexusmods.com/stardewvalley/mods/20659
